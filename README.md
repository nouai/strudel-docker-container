# strudel-docker-container
Simple docker container with Strudel REPL inside

# Installation
```
sudo chmod +x ./init.sh ; ./init.sh
```

# Build
```
./bake.sh
```

# Run container
```
./taste.sh
```

# Stop container
```
./chill.sh
```

# Cleanup / Uninstall
```
./clean.sh
```
